# Asteroid-Activity-Classification
Machine learning workflow for classifying asteroid activity from astronomical observation data, with emphasis on large-scale data processing, model experimentation, and HPC-based training optimization.

This repository contains a machine learning workflow for classifying asteroid activity from astronomical data. The project involved large-scale dataset processing, model training, experimentation, and optimization of compute workflows in an HPC environment. It highlights Python development, machine learning research, parallel computing, and scientific data analysis.
