# Asteroid-Activity-Classification
Machine learning workflow for classifying asteroid activity from astronomical observation data, with emphasis on large-scale data processing, model experimentation, and HPC-based training optimization.

This repository contains a machine learning project focused on classifying asteroid activity from astronomical observation data. The work involves large-scale data processing, model experimentation, and optimization of training workflows in high-performance computing environments.

The project highlights Python-based research computing, machine learning development, parallelization, and technical problem solving in an astroinformatics setting.
